# Neptune SAP Demo - Optimized Appointment Booking

## Overview
Interactive prototype demonstrating an optimized SAP UI for CCR appointment booking system.

## Key Features
- Universal search (address, FLOC, reference, phone)
- Real-time installer availability calendar
- Account tasks and holds management
- Editable contact information
- Automated note generation
- Supervisor instant messaging
- Ticket raising for fully booked schedules

## Impact
- Reduces booking process from 15+ clicks to 3 clicks
- Decreases average appointment time from 10 minutes to 30 seconds
- Saves 9.5 minutes per call

## Tech Stack
- React
- Lucide React (icons)
- Tailwind CSS

## How to Run
```bash
npm install
npm start
```
